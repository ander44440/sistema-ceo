# Arquitetura

Documentos de arquitetura do sistema. Definidos apenas após a conclusão da Engenharia de Requisitos (Fase 0).

## Conceitos arquiteturais candidatos

* **BCO — Banco de Competências Organizacionais** (registrado na ADR-002, Decisão 4): preservar competências aprendidas ao longo da vida do sistema, com ciclo de maturação antes da incorporação ao patrimônio organizacional.
